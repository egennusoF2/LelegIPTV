Verifica del codice Android e Samsung TV

Android / Android TV

Struttura presente nel repo

  • Package e application ID: com.lelegiptv.player.
  • Manifest con INTERNET, servizio media playback, PiP e LEANBACK_LAUNCHER.
  • network_security_config.xml consente HTTP chiaro, necessario per provider Xtream non HTTPS.
  • WebView con fullscreen HTML5, mixed content, debug su build debuggable e bridge per player esterni.

Samsung Smart TV / Tizen

Pacchetto Web App presente

  • Preparazione Tizen: build/tizen-web/.
  • Widget generato: build/tizen-web/LelegIPTV.wgt.
  • Profilo TV: <tizen:profile name="tv"/>.
  • Versione Tizen richiesta: 6.0.

Nota firma Android

APK release attuale non firmato

L'artefatto trovato e' app-universal-release-unsigned.apk. Per distribuzione stabile bisogna configurare il keystore con XTREAM_KEYSTORE_FILE, XTREAM_KEYSTORE_PASSWORD, XTREAM_KEY_ALIAS e XTREAM_KEY_PASSWORD, oppure firmare manualmente l'APK.

Web browser e PWA

Build

pnpm install
pnpm build

Pubblica il contenuto di dist/ su hosting HTTPS, oppure servilo localmente per test. Per PWA, usa HTTPS o localhost; molti browser non permettono l'installazione PWA da HTTP remoto.

Installazione sul dispositivo

  1. Apri l'URL pubblico o locale in Chrome, Safari, Edge o browser TV compatibile.
  2. Se il browser mostra Installa app o Aggiungi a schermata Home, usa quel comando.
  3. Verifica login, Live TV, VOD, tracce audio e sottotitoli dal dispositivo reale.

Android phone e tablet

Build

pnpm install
pnpm tauri:android:build

Se la shell non trova pnpm, apri un terminale login o aggiungi al PATH la directory in cui e' installato Node/pnpm, per esempio Homebrew su macOS.

Installazione via ADB

adb devices
adb install -r src-tauri/gen/android/app/build/outputs/apk/universal/release/app-universal-release-unsigned.apk

Installazione manuale

  1. Copia l'APK sul telefono o tablet.
  2. Abilita l'installazione da sorgenti sconosciute per l'app con cui apri il file.
  3. Apri l'APK e completa l'installazione.

Per installazione ripetibile su piu dispositivi conviene firmare l'APK con lo stesso keystore. Se installi una build firmata con una chiave diversa da quella gia presente, Android richiede la disinstallazione della vecchia app.

Android TV e Google TV

La stessa app Android e' configurata anche per TV: il manifest espone LEANBACK_LAUNCHER, banner TV e supporto D-pad non obbligatorio.

Installazione via rete

  1. Su TV abilita Opzioni sviluppatore e Debug di rete o ADB debugging.
  2. Dal Mac o PC collegato alla stessa rete esegui:
adb connect IP_DELLA_TV:5555
adb install -r src-tauri/gen/android/app/build/outputs/apk/universal/release/app-universal-release-unsigned.apk

Installazione via USB / file manager

  1. Copia l'APK su chiavetta USB o usa un file manager TV.
  2. Abilita installazione da sorgenti sconosciute.
  3. Installa e avvia LelegIPTV dalla schermata app o dalla sezione TV.

Test obbligatori su TV reale: focus D-pad, fullscreen, audio, sottotitoli, HLS/DASH, ritorno indietro e standby/risveglio.

Samsung Smart TV / Tizen

Samsung Tizen non usa Tauri nativo. La versione TV e' una Web Application pacchettizzata in .wgt a partire da dist/.

Prerequisiti

  • Tizen Studio con TV Extension installata.
  • Samsung Certificate Profile valido, creato da Tizen Studio Certificate Manager.
  • TV Samsung in Developer Mode, sulla stessa rete del computer.
  • Tizen TV 6.0 o superiore, come dichiarato in packaging/tizen/config.xml.

Build e packaging

pnpm install
pnpm build
pnpm tizen:prepare
tizen package -t wgt -s NOME_PROFILO_CERTIFICATO -- build/tizen-web

Nel repo e' gia presente build/tizen-web/LelegIPTV.wgt. Se rigeneri la build, usa sempre il pacchetto appena prodotto e non uno vecchio rimasto nella cartella.

Installazione su TV con CLI

sdb connect IP_DELLA_TV
sdb devices
tizen install -n build/tizen-web/LelegIPTV.wgt -t NOME_DEVICE

Installazione con Tizen Studio

  1. Apri Device Manager e collega la TV.
  2. Assicurati che il certificate profile sia selezionato.
  3. Installa il file LelegIPTV.wgt dal menu package/install.

Tizen non ha i plugin Tauri desktop/mobile. Richieste provider, storage e playback seguono il percorso web del browser TV; per questo vanno testati su firmware Samsung reale.

iOS, iPadOS e macOS

iPhone e iPad

pnpm tauri:ios:dev:device
pnpm tauri:ios:build
  1. Apri src-tauri/gen/apple/ in Xcode se devi firmare, archiviare o esportare.
  2. Usa il team Apple configurato nel progetto: F925SY4K37.
  3. Installa su dispositivo con Xcode Devices, Apple Configurator, TestFlight o export .ipa firmato.

Artefatto trovato: src-tauri/gen/apple/build/LelegIPTV.ipa.

macOS

pnpm build:mac
  1. Apri il .dmg generato in src-tauri/target/release/bundle/dmg/.
  2. Trascina LelegIPTV.app in /Applications.
  3. Se Gatekeeper blocca l'app non notarizzata, apri Impostazioni di Sistema > Privacy e Sicurezza e scegli Apri comunque.

Alternativa da terminale per build locali non notarizzate:

xattr -dr com.apple.quarantine "/Applications/LelegIPTV.app"

Windows e Linux

I pacchetti desktop Tauri sono host-specific: compila Windows su Windows e Linux su Linux, a meno di avere una pipeline CI dedicata.

Windows

pnpm install
pnpm build:desktop

Installa l'artefatto in src-tauri/target/release/bundle/nsis/*.exe o msi/*.msi.

Linux

pnpm install
pnpm build:desktop

Installa il pacchetto adatto alla distribuzione: .deb, .rpm o .AppImage.

Checklist finale prima di distribuire

  1. Rigenera sempre gli artefatti dopo modifiche al codice: non riusare pacchetti vecchi.
  2. Controlla nome app, icona e ID: LelegIPTV e com.lelegiptv.player.
  3. Per Android firma l'APK/AAB con keystore stabile prima di consegnarlo a utenti finali.
  4. Per Samsung firma il .wgt con Samsung certificate profile e installalo su TV reale.
  5. Testa almeno un canale live, un VOD, una serie, sottotitoli, tracce audio, resume e fullscreen per ogni famiglia di dispositivi.
  6. Annota in release il percorso preciso dell'artefatto installato, cosi si evita confusione tra build vecchie e nuove.
